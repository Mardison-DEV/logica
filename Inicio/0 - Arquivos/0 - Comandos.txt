git pull origin main - Comando para sincronizar atualizações de outras máquinas

git pull origin main


Saber o que mudou antes de baixar:


git fetch origin
git log HEAD..origin/main --oneline

!!!Se tiver alterações locais não commitadas, o Git pode bloquear o pull. Opções:
